### 3. 核心统计卡片

```
┌──────────────────────────────────────────────────────┐
│ Does not include funding costs and rebates          │
├─────────────┬─────────────┬────────────────────────┤
│ Total P&L:  │ Total Fees: │ Net Realized:          │
│ $571.83     │ $136.60     │ -$31.79                │
└─────────────┴─────────────┴────────────────────────┘
```

#### 统计指标详解

**1. Total P&L（总盈亏）**
- **定义**：已实现 + 未实现盈亏
- **当前**：$571.83
- **含义**：包括未平仓位的浮动盈亏

**2. Total Fees（总手续费）**
- **当前**：$136.60
- **影响**：直接从盈利中扣除
- **占比**：136.60 / 10000 = 1.37% of capital

**3. Net Realized（净已实现）**
- **定义**：已平仓交易的实际盈亏
- **当前**：-$31.79
- **含义**：虽然总P&L为正，但实际平仓的交易还亏损
- **解释**：说明大部分盈利来自未平仓位

#### Average Leverage: 12.7X
- **意义**：平均使用12.7倍杠杆

#### Average Confidence: 69.9%
- **意义**：AI对自己决策的平均信心度
- **范围**：0-100%
- **69.9%**：中等信心，不是完全确定

- 持仓54H 24M

#### Biggest Loss: -$455.66
