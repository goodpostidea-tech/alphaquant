- **更新频率**：每秒更新
- **背景色**：浅灰色 (#F5F5F5)，与主体区域形成对比

#### 技术实现
```javascript
// 数字滚动组件（推测实现）
const AnimatedNumber = ({ value }) => {
  const digits = value.toString().split('');
  
  return digits.map((digit, index) => (
    <span key={index} className="rolling-digit">
      {digit}
    </span>
  ));
};

// CSS动画
.rolling-digit {
  display: inline-block;
  transition: transform 0.3s ease;
}
```


**另一个示例**：
```
┌─────────────────────────────────────────────────┐
│ 🟣 DeepSeek completed a long trade on 💧 XRP!  │
│                                  10/22, 4:01 PM │

└─────────────────────────────────────────────────┘
```

### 数据字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| **模型** | AI模型名称 | DeepSeek Chat V3.1 |
| **方向** | long（做多）/ short（做空） | long |
| **币种** | 交易的加密货币 | XRP |
| **价格** | 入场→出场价格 | $2.4666 → $2.3397 |
| **数量** | 交易数量 | 3542.00 |
| **名义金额** | 交易金额（含杠杆） | $8,737 → $8,287 |
| **持仓时间** | 从开仓到平仓的时长 | 61H 38M |
| **NET P&L** | 净盈亏（扣除手续费） | -$455.66 |
| **时间戳** | 交易完成时间 | 10/23, 5:39 AM |

### 颜色编码
- 🟢 **绿色**：盈利交易、做多方向
- 🔴 **红色**：亏损交易、做空方向

### 关键观察

#### 策略风格差异
- **GPT 5**：多空混合，6个仓位分散
- **Qwen3 Max**：极端集中，全仓单一BTC
- **DeepSeek**：6个仓位，全部做多

#### 杠杆使用
- **最高**：25X（GPT 5的ETH short）
- **常见**：10X-20X
- **风险**：高杠杆意味着小幅波动即可能爆仓

#### 名义价值（Notional）
- **定义**：实际控制的资金量 = 保证金 × 杠杆
- **示例**：$10,000账户，20X杠杆 → 可控制$200,000名义价值

```javascript
useEffect(() => {
  const fetchData = async () => {
    const [prices, accounts, trades, positions] = await Promise.all([
      fetch('/api/crypto-prices'),
      fetch('/api/account-totals'),
      fetch('/api/trades?limit=100'),
      fetch('/api/positions')
    ]);
    
    updateState({ prices, accounts, trades, positions });
```
